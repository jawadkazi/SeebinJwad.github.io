Gaussian Elimination/ Row Reduction

Development of Gaussian elimination methodology, implemented in Python. The program was a final research project for M340L linear algebra, and was built robustly for any valid and real matrix. Additionally, the project was accompanied by a brief research document outlining the pseudo-code for the program, the code itself (which can alternatively be found on my github) and a couple edge cases and future steps in building a more efficient system.

Divbeta.com

After mentoring dozens of pre-seed startups in the Austin area and becoming intimately familiar with the startup environment, I built a tool to help startups solve one key problem in their earliest stages: finding diverse beta testers. The tool was built with Javascript and Flask, has built from scratch login/user management as well as SMTP (simple mail transfer protocol), and has devOps contained within the Render cloud and Github, for convenient and automated CI/CD. This project was then handed to the startup FoundingTitans, who maintain it and collect more users on the platform.
